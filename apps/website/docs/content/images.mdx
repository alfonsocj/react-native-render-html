## Scaling

<p>The renderer will automatically scale images down to the available width, even when the provided inline style width is greater than the container width.</p>

:::tip Tip

You are strongly advised to provide a <a href="https://google.com?q=react-native-render-html%20contentWidth">contentWidth</a> property from <a href="https://reactnative.dev/docs/useWindowDimensions">useWindowDimensions</a> official hook to help this component handle the scaling.
    
:::

## Preloading

<p>Similarly to browsers, this library will place a print box before fetching image dimensions when both <a href="https://mdn.io/attribute/width">width</a> and <a href="https://mdn.io/attribute/height">height</a> attributes are provided, or the two dimensions are set in the <a href="https://mdn.io/attribute/style">style</a> attribute. This is great to avoid images &quot;jumping&quot; from zero height to their computed height, and is a hint to good web design.</p>
